Lua Common Role
============

This role installs ruby 2.0 from the brightbox repository and the 
bundler gem. It sets ruby 2.0 as the default.

Requirements
------------

This role requires Ansible 1.4 or higher, and only has been tested on 
Ubintu 12.04.
